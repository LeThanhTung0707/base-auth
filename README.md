# Fullstack Auth App (NestJS + NextJS)

## Quick start (dev)

### 1) Backend (NestJS)

    cd backend
    cp .env.example .env
    npm install
    npm run start:dev

### 2) Frontend (NextJS)

    cd frontend
    cp .env.example .env
    npm install
    npm run dev

## Notes

- Backend: http://localhost:4000
- Frontend: http://localhost:3000
